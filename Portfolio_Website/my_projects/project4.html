ToDo Planner

A desktop application designed to organize your tasks efficiently.

Project Description

ToDo Planner is a desktop productivity application that helps users organize their tasks, deadlines, and projects with ease. Designed with simplicity in mind, the app provides a clean interface for creating, updating, and tracking tasks. Whether planning daily activities or long-term projects, ToDo Planner enables users to stay focused, organized, and productive.

GitHub Repo

Key Technologies

  • Frontend (UI): Java Swing
  • Backend: Java
  • Database: MySQL + JDBC

Key Features

  • Task & Project Management: Create, edit, and delete tasks with full CRUD functionality.
  • Priorities & Deadlines: Organizes upcoming tasks to help users stay on schedule.
  • Clean, Simple UI: Easy-to-navigate interface designed for efficient task planning.
