IDNAC Travels

A web application designed as an all-in-one travel booking platform.

Project Description

IDNAC Travels is a travel management platform that allows users to book flights and hotels while enabling administrators to manage schedules and availability efficiently. The system provides a secure, user-friendly experience, combining seamless booking, real-time updates, and access control for both users and admins.
This project is still in progress, and has not been completed.

GitHub Repo

Key Technologies

  • Frontend: HTML, CSS
  • Backend: Node.js, JavaScript, TypeORM
  • Database: MySQL

Key Features

  • Secure Authentication: Token-based login with hashed passwords.
  • Access Control: Auth Guards restrict access based on user roles.
  • Booking Management: Users can book, view, and manage flights and hotel reservations.
  • Data-Rich Records: Tracks user, flight, hotel, and token information for transparency and reliability.
